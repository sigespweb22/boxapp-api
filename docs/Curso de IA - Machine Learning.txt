Aprendizagem de máquina Supervisionada(Vem de previsão - Baseado em um conjunto de atributos se faz uma previsão do que pode acontecer no futuro)
MÉTODOS PREDITIVOS

-> Classificação - Na classificação nós temos Rótulos enquanto na regressão nós temos números
    * Nesse modelo sempre envolve uma base de dados histórica
    
    Atributos: Previsores, metas (classes)
     -> Cada tristro pertence a uma classe e possui um conjunto de atributos Previsores.
     -> Objetiva-se descobrir um relacionemnto entre os atributos previsores eo atributo meta.
     -> O  valor do atributo meta é conhecido (aprendizagem supervisionada)

Duas grandes áreas de dados em IA

Duas grandes variáveis:

Numéricas
    -> Numéricas:
        -> Contínua: Armazenam números reais (Exe.: .4, 44.3) (Temperatura, altura, peso) - Float
        -> Discreta: Conjunto de valores finitos (Números inteiros) - Contagem de alguma coisa
        
Categoricas (String)
    -> Nominal: São os dados não mensuráveis e sem ordenação (Exe.: Cor dos olhos, gênero)
    -> Ordinal: Categorização sob ordenação (Exemplo.: Tamanho P, M, G)
